# 04 - Content Strategy

## Mission

I use content to make the product, beliefs, and founder journey visible to the right people.

I want to write refreshing, thought-provoking but highly actionable ideas and stories that leave people empowered to build their business/startup via their own personal brand.

Reach matters, but the real goal is trust, relationships, leads, product learning, and proof.

## Actionable Advice

What ideas can I share to help my One True Fan win?

| Category | Ideas |
| --- | --- |
| Do this | Capture audience language from comments, DMs, calls, and customers. Write from lived proof. Use AI for leverage. Turn product lessons into posts. |
| Stop doing this | Stop chasing generic inspiration. Stop using AI to replace taste. Stop treating content as separate from sales. |
| Common mistakes | Building too much before getting close to customers. Posting without a CTA lane. Trying to win every platform at once. |
| Simple frameworks | Idea -> post -> comment -> DM -> product insight. Audience language -> post angle -> conversation -> offer. |
| Tools or workflows | Impression, content queues, weekly review, comment capture, lead magnet loops, repurposing winners. |
| Stories that teach the lesson | Animation competition, calisthenics transformation, leaving the safe default path, building the wrong version first. |

## Operating Loop

1. Pull one idea from product work, customer conversations, comments, DMs, or old notes.
2. Choose the pillar.
3. Write the useful lesson first.
4. Add proof, story, or example.
5. Publish or schedule.
6. Reply and DM while the post is warm.
7. Save audience language and winning angles for reuse.
