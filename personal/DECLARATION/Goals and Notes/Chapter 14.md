# Workflow / Organization

### Seasons

- Every 3 months create a new chapter page. This is your home page for this period. From here, edit any of the top toggle headings.
- Coloured blocks imply they should *definitely* be edited for this season

### Calendar

- Calendar block me-time every single morning. "I've got plans"
- You cannot waste time as long as you are intentional about how you are spending it.

### Todoist

- Tasks start in inbox (no section). This must stay as clean as possible (no tasks without dates). Scheduled tasks are moved to scheduled section.

Sections:

- Today is most important.
- Up next - sorted by priority. This is where I drag tasks that I definitely need to do, but do not have a specific date (probably because I’m not sure how long they will take)
- Someday - dream list that I can pull from when Up Next is depleted.
- Todoist syncs with G cal in Inbox, and repeat projects for reminders.

### Journal

- Weekly retro/review similar methodology - reminder in todoist.
