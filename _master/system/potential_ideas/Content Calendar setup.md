### **  
Required Plugins**

1. **BRAT** (Community)
    
2. **Full Calendar** (Community)
    
3. **Obsidian Calendar Bases** (Install via BRAT)
    

### **Setup Instructions**

1. **Enable Plugins:**
    
    - Settings > Community Plugins > Browse.
        
    - Install and Enable **BRAT** and **Full Calendar**.
        
2. **Install Beta Plugin:**
    
    - Settings > BRAT > Add Beta plugin.
        
    - Paste the GitHub URL for **Obsidian-Calendar-Bases** > Add Plugin.
        
3. **Configure View:**
    
    - Open your **Base**.
        
    - Click the **Filter** icon > Add folders (e.g., Articles, Emails).
        
    - Click the **View Name** (e.g., "Table") > Configure view.
        
    - Set **Layout** to Calendar.
        
4. **Set Dates:**
    
    - Click **Date properties**.
        
    - Select your **Start date** property (e.g., "date").
